# Детальный план занятия - Массивы и работа с ними в JavaScript

## Цели и задачи

### Основные цели:
- Познакомить студентов с основами работы с массивами в JavaScript
- Научить использовать основные методы массивов для обработки данных
- Развить навыки функционального программирования с массивами
- Объяснить различия между различными методами массивов

### Задачи обучения:
- Уметь объявлять массивы как литералы
- Уметь выбирать нужную часть массива с помощью методов slice
- Уметь выбирать из массива элементы по условию при помощи filter
- Уметь преобразовывать массивы с помощью методов map, reduce
- Уметь выполнять действие для элементов массива с помощью forEach
- Иметь представление о трех аргументах методов forEach, filter, map

### Критерии успеха:
- Студент может самостоятельно создавать и модифицировать массивы
- Студент правильно использует методы forEach, map, filter, reduce
- Студент понимает разницу между различными методами массивов
- Студент применяет методы slice для работы с частями массивов
- Студент использует методы поиска и сортировки в практических задачах

## Тайминг занятия (90 минут)

### Введение (15 минут)
- **0-5 мин:** Приветствие, организационные моменты, повторение предыдущей темы (объекты)
- **5-10 мин:** Обзор темы занятия, постановка целей, важность массивов в программировании
- **10-15 мин:** Введение в массивы - что такое массивы и зачем они нужны, примеры использования

### Основная часть (60 минут)

#### Блок 1: Создание и базовые операции с массивами (20 минут)
- **15-25 мин:** Теория - создание массивов, получение длины, методы push/pop/shift/unshift
- **25-30 мин:** Практика - Задачи 1 и 2 (пустой массив, массив дней недели)
- **30-35 мин:** Разбор методов модификации массивов, вопросы

#### Блок 2: Функциональные методы массивов (20 минут)
- **35-45 мин:** Теория - forEach, map, filter, reduce, три аргумента функций
- **45-50 мин:** Практика - Задачи 3, 4, 5 (фильтрация дней, квадраты чисел)
- **50-55 мин:** Разбор различий между методами, типичные ошибки

#### Блок 3: Поиск, сортировка и работа с частями массивов (20 минут)
- **55-65 мин:** Теория - slice, sort, indexOf, find, findIndex
- **65-70 мин:** Практика - Задача 6 (сумма элементов), дополнительные примеры
- **70-75 мин:** Разбор интересных фактов, вопросы по методам поиска и сортировки

### Заключение (15 минут)
- **75-80 мин:** Повторение ключевых концепций, интересные факты о массивах
- **80-85 мин:** Ответы на вопросы студентов, разбор сложных моментов
- **85-90 мин:** Резюме занятия, домашнее задание, подготовка к следующему уроку

## Домашнее задание

### Обязательная часть:
1. Создать массив из 10 случайных чисел и отсортировать его по убыванию
2. Написать функцию, которая принимает массив строк и возвращает массив их длин
3. Создать массив объектов "студент" (имя, оценка) и найти студента с максимальной оценкой
4. Реализовать функцию, которая принимает массив чисел и возвращает сумму всех четных чисел

### Дополнительная часть:
- Изучить разницу между slice и splice в деталях
- Попрактиковаться с методами find и findIndex на реальных примерах
- Создать функцию для фильтрации уникальных значений в массиве
- Изучить методы работы с многомерными массивами

### Практическое задание:
Разработать программу для анализа данных о продажах: создать массив объектов "продажа" (товар, количество, цена), рассчитать общую выручку, найти самый продаваемый товар, отфильтровать товары с продажами выше среднего.
