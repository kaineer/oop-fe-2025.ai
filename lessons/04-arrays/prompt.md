# Промпт для создания документов по ролям

## Общий контекст урока:
- **Номер урока:** 4
- **Тема:** "Массивы и работа с ними в JavaScript"
- **Продолжительность:** 90 минут
- **Ключевые концепции:** forEach, map, filter, reduce, методы модификации массивов, сортировка, поиск элементов, slice
- **Интересные факты:** Object.keys() возвращает строковые индексы, sort() изменяет исходный массив vs toSorted() создает новый
- **Цели обучения:** Умение объявлять массивы, работать с методами slice, filter, map, reduce, forEach
- **Примеры кода:** Создание массивов, работа со строками и массивами, получение длины, преобразование данных

**Практические упражнения:**
1. Создать пустой массив
2. Создать массив 7 дней недели
3. Выбрать только выходные дни недели
4. Выбрать только рабочие дни недели
5. Для массива от 1 до 10 получить массив квадратов чисел
6. Для массива от 1 до 10 получить сумму элементов
