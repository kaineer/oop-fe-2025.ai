# Промпт для генерации документов урока 7

## Контекст
Это практический урок по JavaScript и веб-разработке для студентов, изучающих ООП и фронтенд-разработку. Урок продолжает разработку приложения для работы со списком слов кроссворда, начатую в предыдущих уроках.

## Метаданные урока
- **Номер урока**: 7
- **Тема**: Модификация списка, удаление элементов, форма для добавления слов
- **Тип**: practice
- **Продолжительность**: 90 минут
- **Проект**: crossword (редактирование и заполнение кроссворда)

## Текущее состояние проекта
На предыдущем уроке создана базовая структура приложения:
- Классы Word и WordList для работы с данными
- Функции рендеринга для отображения списка слов
- Базовая HTML-структура и стили

## Задача урока
Расширить функциональность приложения, добавив возможность:
1. Удалять слова из списка
2. Добавлять новые слова через форму
3. Обновлять существующие слова

## Технические требования
1. Добавить методы removeWord(word) и updateWord(word, description) в WordList
2. Дополнить функцию renderWord() для отображения кнопки удаления
3. Добавить обработчик события click на список слов для удаления
4. Создать форму для добавления слов в index.html
5. Реализовать обработчик события click в форме для добавления слов
6. Добавить логику выбора слова для редактирования
7. Реализовать переключение режимов формы (добавление/редактирование)

## Ожидаемые результаты
- Требования из урока 5 продолжают соблюдаться
- Пользователь может удалить слово из списка
- Пользователь может добавить слово в список
- Пользователь может обновить существующее слово

## Роли для генерации документов
Используй следующие роли из roles.yaml:
- task_description - для создания описания задачи

Проект разделен на шаги:
