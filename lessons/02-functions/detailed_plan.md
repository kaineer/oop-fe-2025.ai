# Детальный план занятия - Функции в JavaScript

## Цели и задачи

### Основные цели:
- Познакомить студентов с различными способами создания функций в JavaScript
- Научить работать с параметрами и возвращаемыми значениями
- Объяснить концепцию контекста выполнения и методов его управления
- Развить навыки использования современных возможностей функций

### Задачи обучения:
- Уметь объявлять именованные и анонимные функции
- Уметь передавать параметры в функцию и получать возвращаемое значение
- Уметь передавать в функцию указанный контекст
- Понимать разницу между function declaration, function expression и стрелочными функциями
- Уметь использовать деструктуризацию и rest параметры

### Критерии успеха:
- Студент может самостоятельно создавать функции разных типов
- Студент правильно передает параметры и обрабатывает возвращаемые значения
- Студент понимает разницу в поведении this в разных типах функций
- Студент умеет использовать call, apply и bind для управления контекстом
- Студент применяет деструктуризацию и rest параметры в практических задачах

## Тайминг занятия (90 минут)

### Введение (15 минут)
- **0-5 мин:** Приветствие, организационные моменты, повторение предыдущей темы
- **5-10 мин:** Обзор темы занятия, постановка целей, важность функций в программировании
- **10-15 мин:** Введение в функции - что такое функции и зачем они нужны

### Основная часть (60 минут)

#### Блок 1: Создание функций (20 минут)
- **15-25 мин:** Теория - function declaration, function expression, стрелочные функции
- **25-30 мин:** Практика - Задачи 1, 4, 5 (базовые функции, стрелочные, анонимные)
- **30-35 мин:** Разбор отличий между способами создания, вопросы

#### Блок 2: Параметры и возвращаемые значения (20 минут)
- **35-45 мин:** Теория - параметры по умолчанию, деструктуризация, rest параметры
- **45-50 мин:** Практика - Задачи 2, 3, 7 (работа с параметрами, возврат значений)
- **50-55 мин:** Разбор типичных ошибок при работе с параметрами, вопросы

#### Блок 3: Контекст выполнения (20 минут)
- **55-65 мин:** Теория - this в функциях, call, apply, bind, контекст стрелочных функций
- **65-70 мин:** Практика - Задача 6 (композиция функций)
- **70-75 мин:** Разбор примеров потери контекста, способы решения, вопросы

### Заключение (15 минут)
- **75-80 мин:** Повторение ключевых концепций, интересные факты о функциях
- **80-85 мин:** Ответы на вопросы студентов, разбор сложных моментов
- **85-90 мин:** Резюме занятия, домашнее задание, подготовка к следующему уроку

## Критерии оценки

### Формативное оценивание:
- **Активность на занятии** - участие в обсуждениях, задавание вопросов
- **Выполнение практических заданий** - правильность синтаксиса, логика решения
- **Понимание концепций** - способность объяснить разницу между типами функций
- **Работа в парах/группах** - совместное решение задач, помощь товарищам

### Суммативное оценивание:
- **Правильность выполнения задач** - соответствие кода требованиям задания
- **Понимание контекста выполнения** - корректное использование this, call, apply, bind
- **Применение современных возможностей** - использование деструктуризации, rest параметров
- **Читаемость кода** - понятные имена функций, правильное форматирование

### Контрольные точки:
- Задача 1: корректное создание функции sum с двумя параметрами
- Задача 2: правильный вывод информации в консоль с использованием шаблонных строк
- Задача 3: нахождение минимального числа из трех
- Задача 4: создание стрелочной функции с правильным синтаксисом
- Задача 5: создание анонимной function expression (с учетом того, что при присваивании константе функция получает имя)
- Задача 6: корректная реализация композиции функций
- Задача 7: извлечение свойства name из объекта

## Домашнее задание

### Обязательная часть:
1. Создать 3 функции разными способами (declaration, expression, стрелочная)
2. Написать функцию, которая принимает объект пользователя и возвращает строку приветствия
3. Создать функцию, использующую деструктуризацию для извлечения нескольких свойств
4. Реализовать функцию, которая принимает неопределенное количество чисел и возвращает их среднее значение

### Дополнительная часть:
- Изучить разницу между function declaration и function expression в деталях
- Попрактиковаться с call, apply и bind на реальных примерах
- Создать набор функций-утилит для работы с массивами
- Изучить концепцию замыканий и создать пример использования

### Практическое задание:
Разработать простой калькулятор с использованием функций для основных операций (сложение, вычитание, умножение, деление), где каждая операция реализована как отдельная функция.
