# Промпт для создания документов по ролям

## Общий контекст урока:
- **Номер урока:** 2
- **Тема:** "Функции в JavaScript"
- **Продолжительность:** 90 минут
- **Ключевые концепции:** Создание функций, ключевое слово function, стрелочные функции, вызов функций напрямую и через call/apply, сохранение контекста, bind, деструктуризация объектов и массивов в параметрах функций, неопределенное количество аргументов
- **Интересные факты:** Свойства length и name у функций, объявленные по умолчанию arguments и event
- **Цели обучения:** Умение объявлять именованные и анонимные функции, передавать параметры и получать возвращаемые значения, передавать указанный контекст

**Практические упражнения:**
1. Создать функцию, которая принимает два числа и возвращает их сумму
2. Создать функцию, которая принимает имя человека и его возраст и выводит в консоль две строки
3. Создать функцию, которая принимает три числа и возвращает самое маленькое из них
4. Создать стрелочную функцию с именем `sum`
5. Создать анонимную нестрелочную функцию
6. Создать функцию, которая принимает две функции и возвращает композицию функций
7. Создать функцию, принимающую объект и возвращающую значение свойства `name`

## Роль 1: lesson_guide (Краткий конспект)

**Аудитория:** Преподаватель
**Цель:** Структура и ключевые моменты для ведения занятий
**Тон:** Лаконичный, только факты
**Файл:** `lesson_guide.md`

**Структура документа:**
1. **Ключевые концепции** - создание и вызов функций в JavaScript
2. **Примеры для демонстрации** - практические примеры из упражнений
3. **Интересные факты** - необычные особенности функций для вовлечения студентов
4. **Вопросы для обсуждения** - темы для вовлечения студентов
5. **Типичные ошибки** - распространенные проблемы новичков

**Требования к содержанию:**
- Использовать только факты из метаданных
- Быть максимально лаконичным
- Сосредоточиться на том, что действительно нужно преподавателю для ведения занятия
- Включить проверочные тесты из упражнений
- Предложить дополнительные вопросы для проверки понимания
- Включить интересные факты для вовлечения студентов в обсуждение
