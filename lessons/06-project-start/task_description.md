# Описание задачи - Создание проекта кроссворда

## Обзор проекта

В этом практическом занятии вы создадите свой первый веб-проект - редактор кроссворда. Вы научитесь создавать проект с нуля, настраивать окружение разработки, работать с объектами JavaScript, организовывать код в модули и отображать данные на веб-странице.

## Ожидаемые результаты

После успешного выполнения проекта вы сможете продемонстрировать:

1. **Проект создается и отображается в браузере** - веб-приложение корректно запускается и отображает базовый интерфейс
2. **Создаются объекты для поддержки списка используемых слов** - реализованы классы для работы со словами и их списками
3. **Список используемых слов отрисовывается на странице** - данные из объектов корректно отображаются в пользовательском интерфейсе

## Требования к решению

### Функциональные требования:
1. Создать проект с использованием команды `npm create`
2. Настроить базовую структуру файлов
3. Создать классы для представления слов и списка слов
4. Организовать код в модульной структуре
5. Реализовать функции отрисовки слов и списка слов
6. Отобразить список слов на веб-странице

### Технические требования:
- Использовать Vanilla JavaScript (без фреймворков)
- Проект должен запускаться командой `npm run dev`
- Код должен быть читаемым и хорошо структурированным
- Использовать современные возможности JavaScript (ES6+ модули)
- Реализовать модульную организацию кода

## Элементы языка, которые нужно использовать

### Команды npm:
```bash
npm create
npm install
npm run dev
```

### Объекты и классы:
```javascript
// Создание классов
class Word {
    constructor(word, description) {
        this.word = word;
        this.description = description;
    }
}

class WordList {
    constructor() {
        this.words = [];
    }
    
    addWord(word, description) {
        this.words.push(new Word(word, description));
    }
}
```

### Функции отрисовки (возвращают HTML строку):
```javascript
// Функция отрисовки одного слова (возвращает HTML строку)
function renderWord(word) {
    return `
        <div class="word-item">
            <strong>${word.word}</strong>: ${word.description}
        </div>
    `;
}

// Функция отрисовки списка слов (возвращает HTML строку)
function renderWordList(wordList) {
    return wordList.words
        .map(word => renderWord(word))
        .join('');
}
```

### Модули ES6:
```javascript
// Экспорт классов и функций
export class Word {
    // ...
}

export function renderWord(word) {
    // ...

export function renderWordList(wordList) {
    // ...
}

// Импорт модулей
import { Word } from "./word.js";
import { renderWordList } from "./renderers.js";
```

### Работа с DOM:
```javascript
// Добавление HTML в DOM
document.getElementById('word-list').innerHTML = renderWordList(wordList);
```

### Работа с файлами:
- Изменение HTML документа

- Организация структуры проекта
- Создание модульных файлов
- Реализация функций отрисовки

## Этапы реализации

### Этап 1: Создание проекта
1. Выполните команду: `npm create`
2. Выберите опцию "Vanilla" и "JavaScript"
3. Установите зависимости: `npm install`
4. Запустите проект: `npm run dev`

### Этап 2: Настройка структуры
1. Измените заголовок документа в `index.html`
2. Удалите лишние файлы из папок `public/` и `src/`
3. Очистите основной JavaScript файл `main.js` от лишнего кода
4. Удалите стили из `style.css`

### Этап 3: Создание объектов для слов
1. Создайте файл `src/word.js` с классом `Word`
2. Добавьте свойства: `word` и `description`
3. Создайте файл `src/word-list.js` с классом `WordList`
4. Реализуйте метод `addWord` для добавления слов в список

### Этап 4: Создание функций отрисовки
1. Создайте файл `src/renderers.js` с функциями отрисовки
2. Реализуйте функцию `renderWord` для отрисовки одного слова (возвращает HTML строку)
3. Реализуйте функцию `renderWordList` для отрисовки всего списка (возвращает HTML строку)
4. Настройте импорт/экспорт между модулями

### Этап 5: Отрисовка списка слов
1. Импортируйте необходимые модули в `main.js`
2. Создайте экземпляр `WordList` и добавьте тестовые слова
3. Используйте `innerHTML` для отображения HTML строки, возвращаемой функцией `renderWordList`
4. Протестируйте отображение созданных слов

## Пример ожидаемого результата

После выполнения всех этапов у вас должно получиться веб-приложение, которое:
- Успешно запускается командой `npm run dev`
- Отображает базовую страницу с заголовком "Crossword"
- Имеет модульную структуру с классами Word и WordList
- Содержит функции отрисовки, возвращающие HTML строки
- Отображает список слов с их описаниями на странице
- Готово для дальнейшей разработки функциональности кроссворда
