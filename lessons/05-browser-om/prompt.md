# Промпт для создания документов по ролям

## Общий контекст урока:
- **Номер урока:** 5
- **Тема:** "Объектная модель браузера"
- **Продолжительность:** 90 минут
- **Ключевые концепции:** Методы window, document, navigator, querySelector, querySelectorAll, children, parentElement, closest, className, classList, dataset, append, remove
- **Интересные факты:** jQuery собирается выпустить 4ю версию
- **Цели обучения:** Умение получать доступ к элементам страницы, изменять стили элементов, добавлять и удалять элементы на странице
- **Примеры кода:** Получение доступа к элементам по id, классу и тегу, работа с querySelector, изменение классов и стилей, удаление элементов

**Практические упражнения:**
1. Получаем ссылку на элемент в списке
2. Получаем ссылку на все элементы списка
3. Получаем ссылку на tr если есть ссылка на td
4. Считаем количество дочерних элементов

**Требования к содержанию:**
- Использовать только факты из метаданных
- Быть максимально лаконичным
- Сосредоточиться на том, что действительно нужно преподавателю для ведения занятия
