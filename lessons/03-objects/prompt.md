# Промпт для создания документов по ролям

## Общий контекст урока:
- **Номер урока:** 3
- **Тема:** "Объекты в JavaScript"
- **Продолжительность:** 90 минут
- **Ключевые концепции:** Создание объектов как литералов, через конструкторы, Object.create(), классы, наследование, геттеры и сеттеры
- **Интересные факты:** null это тоже объект, прототипирование дает выигрыш в производительности при большом количестве объектов
- **Цели обучения:** Умение объявлять объекты как литералы, создавать классы, работать с наследованием
- **Примеры кода:** Создание объектов как литералов, через конструкторы, Object.create(), классы и наследование, объекты с геттерами

**Практические упражнения:**
1. Создать пустой объект
2. Создать объект с методом, который будет возвращать свойство объекта (например, метод getName())
3. Создать класс Point с полями x и y
4. Добавить в класс Point метод move(newX, newY)
5. Добавить в класс Point метод moveBy(shiftX, shiftY)
6. Добавить в класс Point метод mirror(zeroX, zeroY)
