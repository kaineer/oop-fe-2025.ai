# Промпт для создания документов по ролям

## Общий контекст урока:
- **Номер урока:** 1
- **Тема:** "Типы данных в JavaScript"
- **Продолжительность:** 90 минут
- **Ключевые концепции:** Ключевые слова для объявления переменных, строки, числа, логические значения
- **Интересные факты:** Ошибка 0.1 + 0.2, деление на ноль, число которое не равняется себе
- **Цели обучения:** Умение объявлять переменные, работать с числами, использовать операторы сравнения

**Практические упражнения:**
1. Объявление переменных (переменная `name` со значением 'Василий')
2. Сложение чисел (5 + 3 = 8)
3. Сравнение строк ('apple' === 'banana' → false)

---

## Роль 1: lesson_guide (Краткий конспект)

**Аудитория:** Преподаватель
**Цель:** Структура и ключевые моменты для ведения занятий
**Тон:** Лаконичный, только факты
**Файл:** `lesson_guide.md`

**Структура документа:**
1. **Ключевые концепции** - основные типы данных в JavaScript
2. **Примеры для демонстрации** - практические примеры из упражнений
3. **Интересные факты** - необычные особенности JavaScript для вовлечения студентов
4. **Вопросы для обсуждения** - темы для вовлечения студентов
5. **Типичные ошибки** - распространенные проблемы новичков

**Требования к содержанию:**
- Использовать только факты из метаданных
- Быть максимально лаконичным
- Сосредоточиться на том, что действительно нужно преподавателю для ведения занятия
- Включить проверочные тесты из упражнений
- Предложить дополнительные вопросы для проверки понимания
- Включить интересные факты для вовлечения студентов в обсуждение

---

## Роль 2: full_notes (Полный конспект)

**Аудитория:** Студенты (пропустившие занятие)
**Цель:** Полное объяснение темы для тех, кто не смог прийти
**Тон:** Доступный, подробный
**Файл:** `full_notes.md`

**Структура документа:**
1. **Теория с примерами** - подробное объяснение концепций
2. **Пошаговые объяснения** - детальные разборы упражнений
3. **Интересные факты** - необычные особенности JavaScript для лучшего запоминания
4. **Дополнительные материалы** - ссылки на ресурсы, конкретные видео

**Требования к содержанию:**
- Объяснять концепции так, как будто студент видит их впервые
- Включать пошаговые разборы всех практических упражнений
- Предоставлять дополнительные ресурсы для самостоятельного изучения
- Использовать доступный язык, понятный новичкам
- Включить интересные факты для лучшего запоминания материала

---

## Роль 3: detailed_plan (Детальный план занятия)

**Аудитория:** Преподаватель/методист
**Цель:** Описание пройденных тем с почасовкой
**Тон:** Доступный, подробный
**Файл:** `detailed_plan.md`

**Структура документа:**
1. **Цели и задачи** - что студенты должны уметь после занятия
2. **Тайминг (по занятиям и парам)** - распределение времени по темам
3. **Материалы и инструменты** - что понадобится для проведения занятия
4. **Критерии оценки** - как оценивать успеваемость студентов

**Требования к содержанию:**
- Предоставить детальный план с разбивкой по времени
- Указать необходимые материалы и инструменты
- Определить критерии успешного освоения материала
- Включить рекомендации по адаптации плана при необходимости

---

## Общие требования ко всем документам:
- Использовать только факты из метаданных уроков
- Соответствовать тону и аудитории каждой роли
- Быть структурированным и легко читаемым
- Включать все ключевые концепции и упражнения урока
- Сохранять профессиональный подход к изложению материала

Создай три документа в папке `01-javascript-intro` на основе этого промпта: `lesson_guide.md`, `full_notes.md`, и `detailed_plan.md`.
