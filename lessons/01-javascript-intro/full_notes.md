# Полный конспект занятия - Типы данных в JavaScript

## Теория с примерами

### Введение в переменные

Переменные в JavaScript - это контейнеры для хранения данных. Для их объявления используются ключевые слова:

**`let`** - используется для переменных, значения которых могут изменяться:
```javascript
let userName = 'Мария';
userName = 'Анна'; // значение можно изменить
```

**`const`** - используется для констант, значения которых не могут изменяться:
```javascript
const PI = 3.14;
// PI = 3.15; // ОШИБКА! Константу нельзя переопределить
```

**`var`** - устаревшее ключевое слово, используется для обратной совместимости. Отличается от `let` областью видимости и поведением:
```javascript
var oldVariable = 'устаревший способ';
// Не рекомендуется использовать в современном коде
```

### Основные отличия var от let/const

- **Область видимости:** `var` имеет функциональную область видимости, а `let`/`const` - блочную
- **Поднятие (hoisting):** `var` поднимается со значением `undefined`, `let`/`const` тоже поднимаются, но не могут использоваться до объявления
- **Переопределение:** `var` можно переопределять в той же области видимости

**Рекомендация:** Используйте `let` и `const` в современном JavaScript, `var` - только для поддержки старого кода.

### Основные типы данных

#### Строки (String)
Строки представляют текстовые данные и заключаются в кавычки:
```javascript
let greeting = 'Привет, мир!';
let name = "Алексей";
```

#### Числа (Number)
Числа могут быть целыми или дробными:
```javascript
let age = 25;
let price = 99.99;
let temperature = -5;
```

#### Логические значения (Boolean)
Логические значения могут быть только `true` (истина) или `false` (ложь):
```javascript
let isActive = true;
let isCompleted = false;
```

### Арифметические операции

С числами можно выполнять различные математические операции:
```javascript
let sum = 5 + 3;        // сложение: 8
let difference = 10 - 4; // вычитание: 6
let product = 4 * 2;     // умножение: 8
let quotient = 15 / 3;   // деление: 5
```

### Операторы сравнения

Операторы сравнения возвращают логические значения:
```javascript
let isEqual = 5 === 5;        // true
let isNotEqual = 5 === 3;     // false
let isGreater = 10 > 5;       // true
let isLess = 3 < 2;           // false
```

## Пошаговые объяснения упражнений

### Задача 1: Объявление переменных

**Задача:** Объявите переменную `name` и присвойте ей значение 'Василий'.

**Пошаговое решение:**
1. Используем ключевое слово `let` для объявления переменной
2. Даем переменной понятное имя `name`
3. Присваиваем значение 'Василий' с помощью оператора `=`
4. Закрываем инструкцию точкой с запятой

```javascript
let name = 'Василий';
```

**Проверка:**
```javascript
console.log(name); // выведет: Василий
console.log(name === 'Василий'); // выведет: true
```

### Задача 2: Сложение чисел

**Задача:** Сложите числа 5 и 3 и присвойте результат переменной `result`.

**Пошаговое решение:**
1. Объявляем переменную `result`
2. Используем оператор сложения `+` для вычисления 5 + 3
3. Результат автоматически присваивается переменной

```javascript
let result = 5 + 3;
```

**Проверка:**
```javascript
console.log(result); // выведет: 8
console.log(result === 8); // выведет: true
```

### Задача 3: Сравнение строк

**Задача:** Сравните строки 'apple' и 'banana' и присвойте результат переменной `comparison`.

**Пошаговое решение:**
1. Объявляем переменную `comparison`
2. Используем оператор строгого равенства `===`
3. Сравниваем две разные строки
4. Результат сравнения (false) присваивается переменной

```javascript
let comparison = 'apple' === 'banana';
```

**Объяснение результата:**
- Строки 'apple' и 'banana' разные
- Оператор `===` проверяет точное равенство
- Поэтому результат сравнения - `false`

**Проверка:**
```javascript
console.log(comparison); // выведет: false
console.log(comparison === false); // выведет: true
```

## Интересные факты о JavaScript

### Ошибка 0.1 + 0.2, которая ошибкой не является
В JavaScript, как и во многих других языках программирования, числа с плавающей точкой представлены в двоичной системе. Из-за этого некоторые десятичные дроби не могут быть точно представлены:
```javascript
console.log(0.1 + 0.2); // выведет: 0.30000000000000004
console.log(0.1 + 0.2 === 0.3); // выведет: false
```
Это не ошибка JavaScript, а особенность стандарта IEEE 754 для чисел с плавающей точкой.

### Внезапно, деление на ноль
В отличие от многих языков программирования, деление на ноль в JavaScript не вызывает ошибку:
```javascript
console.log(1 / 0); // выведет: Infinity
console.log(-1 / 0); // выведет: -Infinity
console.log(0 / 0); // выведет: NaN
```

### Число, которое не равняется себе
`NaN` (Not a Number) - это специальное значение, которое представляет результат неопределенной или недопустимой математической операции. Уникальное свойство `NaN` в том, что оно не равно даже самому себе:
```javascript
console.log(NaN === NaN); // выведет: false
console.log(isNaN(NaN)); // выведет: true (используйте эту функцию для проверки)
```
