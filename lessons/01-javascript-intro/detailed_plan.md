# Детальный план занятия - Типы данных в JavaScript

## Цели и задачи

### Основные цели:
- Познакомить студентов с основами JavaScript
- Научить работать с переменными и типами данных
- Развить навыки выполнения арифметических операций
- Обучить использованию операторов сравнения

### Задачи обучения:
- Уметь объявлять переменные и назначать им значения
- Уметь работать с числовыми значениями и выполнять над ними арифметические операции  
- Уметь использовать операторы сравнения и логические операторы
- Понимать разницу между `let`, `const` и `var`

### Критерии успеха:
- Студент может самостоятельно объявить переменные разных типов
- Студент выполняет базовые арифметические операции
- Студент правильно использует операторы сравнения
- Студент понимает разницу между `let`, `const` и `var`

## Тайминг занятия (90 минут)

### Введение (15 минут)
- **0-5 мин:** Приветствие, организационные моменты
- **5-10 мин:** Обзор темы занятия, постановка целей
- **10-15 мин:** Введение в переменные и типы данных, включая `var`

### Основная часть (60 минут)

#### Блок 1: Объявление переменных (20 минут)
- **15-25 мин:** Теория - ключевые слова `let`, `const` и `var`
- **25-30 мин:** Практика - Задача 1 (Объявление переменных)
- **30-35 мин:** Разбор типичных ошибок, включая использование `var`, вопросы

#### Блок 2: Числа и арифметика (20 минут)
- **35-45 мин:** Теория - числовые типы, арифметические операции
- **45-50 мин:** Практика - Задача 2 (Сложение чисел)
- **50-55 мин:** Дополнительные примеры, вопросы

#### Блок 3: Сравнение и логические значения (20 минут)
- **55-65 мин:** Теория - операторы сравнения, логические значения
- **65-70 мин:** Практика - Задача 3 (Сравнение строк)
- **70-75 мин:** Разбор результатов, вопросы

### Заключение (15 минут)
- **75-80 мин:** Повторение ключевых концепций, интересные факты
- **80-85 мин:** Ответы на вопросы студентов
- **85-90 мин:** Резюме занятия, домашнее задание

## Критерии оценки

### Формативное оценивание:
- **Активность на занятии** - участие в обсуждениях, вопросы
- **Выполнение практических заданий** - правильность кода
- **Понимание концепций** - способность объяснить свои решения

### Суммативное оценивание:
- **Правильность выполнения задач** - соответствие кода требованиям
- **Понимание типов данных** - умение работать со строками, числами, логическими значениями
- **Применение операторов** - корректное использование арифметических и логических операторов

### Контрольные точки:
- Задача 1: правильное объявление переменной `name` со значением 'Василий'
- Задача 2: корректное вычисление суммы 5 + 3
- Задача 3: правильное сравнение строк 'apple' и 'banana'

## Домашнее задание

### Обязательная часть:
1. Объявить 3 переменные разных типов (строка, число, логическое значение)
2. Выполнить 2 арифметические операции с числами
3. Сравнить 2 пары значений разных типов

### Дополнительная часть:
- Изучить разницу между `==` и `===`
- Попрактиковаться с операторами `>=` и `<=`
- Создать простой калькулятор для 4 основных операций
