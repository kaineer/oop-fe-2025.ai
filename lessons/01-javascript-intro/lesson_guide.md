# Краткий конспект занятия - Типы данных в JavaScript

## Ключевые концепции

### Ключевые слова для объявления переменных
- `let` - для переменных, которые могут изменяться
- `const` - для констант (неизменяемых значений)
- `var` - устаревшее ключевое слово, используется для обратной совместимости

### Основные типы данных
- **Строки** - текстовые данные в кавычках (`'текст'` или `"текст"`)
- **Числа** - целые и дробные числа (`5`, `3.14`)
- **Логические значения** - `true` или `false`

## Примеры для демонстрации

### Объявление переменных
```javascript
let name = 'Василий';
```

### Сложение чисел
```javascript
let result = 5 + 3; // результат: 8
```

### Сравнение строк
```javascript
let comparison = 'apple' === 'banana'; // результат: false
```

## Интересные факты

- **Ошибка 0.1 + 0.2:** В JavaScript `0.1 + 0.2` не равно `0.3` из-за особенностей представления чисел с плавающей точкой
- **Деление на ноль:** `1 / 0` возвращает `Infinity`, а не вызывает ошибку
- **Число, которое не равняется себе:** `NaN` (Not a Number) не равно даже самому себе: `NaN === NaN` возвращает `false`

## Типичные ошибки

- Использование переменных без объявления
- Путаница между одинарными и двойными кавычками
- Попытка сложить число и строку (результат - конкатенация)
- Использование `=` вместо `===` для сравнения
- Переопределение констант (`const`)
- Использование `var` вместо `let`/`const` в современном коде

**Описание:** Сложите числа 5 и 3 и присвойте результат переменной `result`.
